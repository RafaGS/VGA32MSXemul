# VGA32MSXemul

----

MSX Emulator for FabGL cards

-----

This is an MSX1 emulator for ESP32 systems compatible with FabGL/VGA32.

Installation and usage:

- For [Samsung SPC-800](https://minibots.wordpress.com/2026/07/12/emulador-de-samsung-spc-800-con-lilygo-ttgo-vga32/)

- For [National CF-2000](https://minibots.wordpress.com/2026/07/16/emulador-de-national-cf-2000-con-olimex-esp32-sbc-fabgl/)
